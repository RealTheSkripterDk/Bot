Hvad er Fantasy

Fantasy er en minecraft server, som er lavet TixDk og TheSkripterDk. Vi lavet serveren Fantasy fordi vi vil lave en server som var få alle og ikke som andre. Vi startet med og lave en smp server som vi også har endnu. men så fik vi ideen om og lave det til en hub server.

Vores ide med serveren er og prøve og lave servers som ikke er sådan meget set før i minecraft verden men det kan godt være vi ikke er lige så stor som FV og SA men det er ikke det vi tinger på vi tinger på at lave en minecraft som ikke er set før i minecraft verden.
